Help Center

Setup, workflow, and support for Short Video Publisher.

This help center explains how creators move from the public website into the publisher workspace, connect TikTok, review posting settings, send either Direct Post or Upload fallback, and complete any remaining TikTok steps.

Getting Started

1

Open the workspace

Move from the public website into the dedicated publisher workspace page.

2

Connect TikTok

Use Login Kit to authorize the creator account and return to the workspace.

3

Load creator settings and send

Review privacy, interaction toggles, and duration limits before Direct Post or Upload fallback.

Current TikTok Scope

  • Login Kit for creator account connection
  • `user.info.basic` for the connected account display
  • `video.upload` for the live Upload API fallback
  • Review-ready Direct Post controls that activate after `video.publish` approval
  • Publish status inspection from the returned publish ID
  • Creator-controlled privacy, interaction, and consent review inside the workspace

Posting Workflow

  • Prepare the public MP4 URL, TikTok title, and internal approval note inside the workspace
  • Load creator settings so privacy options and interaction limits come from TikTok, not from defaults
  • Collect creator consent before sending Direct Post or Upload fallback
  • Keep the publish ID to inspect status later from the same workspace
