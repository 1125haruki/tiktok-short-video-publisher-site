Help Center

Setup, workflow, and support for Short Video Publisher.

This help center explains how creators move from the public website into the publisher workspace, connect TikTok, prepare a draft upload, and complete the final publish review on TikTok.

Getting Started

1

Open the workspace

Move from the public website into the dedicated publisher workspace page.

2

Connect TikTok

Use Login Kit to authorize the creator account and return to the workspace.

3

Create a draft upload

Prepare the public MP4 source and initialize the TikTok draft request.

Current TikTok Scope

  • Login Kit for creator account connection
  • `user.info.basic` for the connected account display
  • `video.upload` for TikTok draft upload initialization
  • Publish status inspection from the returned publish ID
  • Final editing and publish confirmation on TikTok

Draft Workflow

  • Prepare the public MP4 URL and internal posting note inside the workspace
  • Confirm creator authorization before creating the draft upload request
  • Keep the publish ID to inspect status later from the same workspace
  • Finish final review in TikTok after the draft is created
